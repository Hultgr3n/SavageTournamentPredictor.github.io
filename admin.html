🌐 WC2026 API

Paste your WC2026 API bearer key from api.wc2026api.com/docs. The site will use it to sync teams, fixtures, and live scores.

Sync Scores Only — updates match results without changing team assignments or fixture structure. Use this after the group stage is set.
Sync Fixtures & Scores (full) — rewrites all match data from the API, including team names. Can corrupt the knockout bracket if the API has wrong data.

🔧 Reset Corrupted Fixture & Diagnostics

The API uses different internal IDs than FIFA's official match numbers, so Sync Scores Only now matches by kick-off time to route scores to the correct bracket doc. Use Diagnose to see exactly which Firestore doc IDs exist for knockout matches — cross-reference with expected bracket positions to spot mismatches. Reset Fixture wipes a single corrupted doc; user predictions are never touched.
